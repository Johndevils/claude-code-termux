Setup Guide

Copy, paste, done. Get running in under 5 minutes.

01

Install Termux

Get the latest Termux from F-Droid (recommended over Play Store).

termux
$pkg update && pkg upgrade -y
$pkg install git nodejs-lts python -y
02

Install Ollama

Set up Ollama to run models locally or connect to a cloud instance.

termux
$curl -fsSL https://ollama.com/install.sh | sh
$ollama serve &
$ollama pull codestral
03

Install Claude Code

Install the Anthropic CLI and configure it with your API key.

termux
$npm install -g @anthropic-ai/claude-code
$export ANTHROPIC_API_KEY=sk-ant-...
04

Connect Ollama to Claude Code

Configure Claude Code to use Ollama as a model provider.

termux
$claude config set model ollama:codestral
$claude config set ollamaBaseUrl http://localhost:11434
05

Launch & Code

You're ready. Start coding with AI assistance.

termux
$mkdir my-project && cd my-project
$claude
✓ Connected to ollama:codestral
✓ Context window: 128k tokens
✓ Ready to code
